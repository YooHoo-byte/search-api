Overview

The Search API is a Node.js-based service built with Express.js that provides unified access to multiple privacy-respecting search engines and content providers. Unlike traditional search APIs, this service prioritizes user privacy, avoids tracking, and offers transparent data aggregation from ethical sources.

Privacy-First

No user tracking or behavior profiling. All searches are anonymous.

High Performance

Intelligent caching, connection pooling, and parallel processing.

Multi-Source

Aggregates results from multiple privacy-respecting providers.

Key Features

Multi-Source Search

  • Web search from DuckDuckGo, StartPage, and more
  • Image search from privacy-focused sources
  • Video search across ethical platforms
  • News aggregation from independent sources
  • Weather data without tracking

Performance

  • Intelligent caching with configurable TTL
  • Automatic fallback between providers
  • Request retry with exponential backoff
  • Connection pooling with keep-alive
  • 100+ results per category

Security

  • No tracking or data collection
  • Configurable safe search filtering
  • XSS protection and input sanitization
  • Security headers with Helmet.js
  • Rate limiting and DDoS protection

Developer Experience

  • Simple RESTful JSON API
  • Comprehensive documentation
  • Extensible provider architecture
  • Environment-based configuration
  • Docker and Kubernetes support

Quick Start

1

Clone the Repository

git clone https://github.com/YooHoo-byte/search-api.git
cd search-api
2

Install Dependencies

npm install
3

Configure Environment

cp .env.example .env
# Edit .env with your configuration
4

Start the Server

npm start
# or for development
npm run dev
5

Test the API

curl "http://localhost:3000/search?q=privacy+technology"

For detailed installation instructions, see the Getting Started guide.
